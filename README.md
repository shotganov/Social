Автор: **Шаганов Егор Юрьевич**.

## О проекте

**Social** — учебная социальная сеть: лента постов, профили, комментарии, лайки, подписки, чаты, поиск и уведомления.

### Задача

Дать пользователям лёгкое пространство для коротких текстовых публикаций и общения: делиться мыслями, читать ленту тех, на кого подписан, обсуждать посты в комментариях и общаться лично в чатах.

### Целевая аудитория

Пользователи, которым нужна минималистичная альтернатива большим соцсетям — без рекламы и алгоритмических лент: студенты, авторы коротких заметок, сообщества по интересам.

### Возможности

- Регистрация и вход по логину/паролю (JWT-аутентификация).
- Лента постов с лайками и комментариями.
- Создание поста с текстом и изображениями, счётчик символов, эмодзи-пикер.
- Просмотр и редактирование собственного профиля (аватар, обложка, имя, био).
- Подписки на других пользователей и страница подписок.
- Поиск пользователей и постов.
- Чаты между пользователями.
- Уведомления о действиях.
- Жалобы на посты.
- Адаптивная вёрстка: десктоп, компактный сайдбар, мобильный нижний бар.

## Стек

- **Frontend:** Vite, React 19, TypeScript, MUI, Redux Toolkit, React Query, React Router, Feature-Sliced Design.
- **Backend:** NestJS 11, Prisma 5, PostgreSQL, JWT (passport-jwt), bcrypt.

## Запуск проекта

Проект — монорепо из двух независимых пакетов: `frontend` и `backend`. Команды запускайте из соответствующей директории.

### Требования

- Node.js ≥ 20
- PostgreSQL ≥ 14
- npm

### 1. Backend

```powershell
cd final-project-2-semester/backend
npm install
```

Создайте файл `.env` в `backend/` со строкой подключения к БД и секретом JWT:

```env
DATABASE_URL="postgresql://USER:PASSWORD@localhost:5432/social"
JWT_SECRET="any-long-secret"
```

Первичная настройка БД (генерация Prisma-клиента + миграции + сидинг тестовых данных):

```powershell
npm run db:setup
```

Если БД уже была — пересоздать с нуля:

```powershell
npm run db:refresh
```

Запуск API в режиме разработки:

```powershell
npm run start:dev
```

API поднимется на `http://localhost:3000`.

### 2. Frontend

```powershell
cd final-project-2-semester/frontend
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.
